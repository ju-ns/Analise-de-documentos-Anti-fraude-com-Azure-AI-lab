# Analise-de-documentos-Anti-fraude-com-Azure-AI-lab
Projeto desenvolvido no Bootcamp DIO, utilizando Azure AI Document Intelligence e Azure Blob Storage para analisar imagens de documentos e identificar possíveis fraudes. A aplicação recebe uma imagem, envia ao Blob Storage, processa com IA e exibe o resultado em uma interface simples feita com Streamlit.
